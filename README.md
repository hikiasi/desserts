# 🍰 Платформа для продажи замороженных десертов в Калининграде

Это полноценный full-stack проект, разработанный на Next.js, для официального дистрибьютора премиальных замороженных десертов. Платформа включает в себя витрину для розничных покупателей, мощный B2B-раздел для кафе и магазинов, а также защищенную админ-панель для управления товарами и заказами.

## 🚀 Основные возможности

### Для B2B клиентов (Кафе, Рестораны, Магазины)
- **Калькулятор маржи:** Интерактивный расчет прибыли в зависимости от объема продаж и наценки.
- **Оптовые цены:** Автоматический пересчет стоимости при заказе от 20 единиц товара.
- **Пробная партия:** Специальная механика первого заказа со скидкой 20%.
- **Процесс работы:** Прозрачный timeline от заявки до регулярных поставок.
- **Кейсы:** Реальные истории успеха партнеров в Калининграде.

### Для розничных покупателей
- **Удобный каталог:** Фильтрация по категориям (Профитроли, ЗОЖ, Торты).
- **Корзина:** Умная корзина с индикатором бесплатной доставки и автоматическим применением оптовых скидок.
- **Простой заказ:** Оформление через форму с интеграцией Telegram-уведомлений.

### Для администратора
- **Управление каталогом:** Добавление, редактирование и удаление товаров.
- **Управление заказами:** Просмотр входящих заявок с разделением на B2B и розницу.
- **Статистика:** Базовые метрики продаж.

## 🛠 Технологический стек

- **Frontend:** Next.js 15 (App Router), React 19, Tailwind CSS, shadcn/ui.
- **Animations:** Framer Motion (производительные анимации для мобильных устройств).
- **Backend:** Next.js API Routes, Prisma ORM.
- **Database:** SQLite (локальное хранилище для максимальной скорости).
- **Validation:** Zod, React Hook Form.
- **Icons:** Lucide React.
- **Maps:** Яндекс.Карты API.

## 📦 Быстрый старт

1. **Установка зависимостей:**
   ```bash
   npm install
   ```

2. **Настройка окружения:**
   Создайте файл `.env` на основе `.env.example` и укажите необходимые ключи.

3. **Инициализация базы данных:**
   ```bash
   npx prisma generate
   npx prisma db push
   npx prisma db seed
   ```

4. **Запуск проекта:**
   ```bash
   npm run dev
   ```

## ❄️ Особенности продукции
- **Шоковая заморозка:** Сохранение вкуса и текстуры до 180 дней при -18°C.
- **Натуральный состав:** Натуральные сливки, сливочное масло 82.5%, отсутствие консервантов.
- **ЗОЖ-линейка:** Десерты без сахара и глютена.
